# Learn How to Make Corewar Champions (DRAGON, MANDOM Method)

The goal of this repository is to learn how to create a Corewar champion that can move backward through memory.

## How to Use

Use the binaries:

```bash
./corewar-mandom-1 <your-.cor>
```

If you want to see it visually:

```bash
./corewar-mandom-1 <your-.cor> -gui
```

You can also use `tester.sh`:

```bash
./tester.sh
```

## How to win

### 1.

Create a champion that moves backward to position 1000 in memory.

### 2.

Create a champion that moves backward to position 1000 in memory.
